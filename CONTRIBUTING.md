# Contributing to Lowvia

First off, thank you for considering contributing to Lowvia! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/emanuelVINI01/lowvia/issues) to see if someone else has already created a ticket. If not, go ahead and [make one](https://github.com/emanuelVINI01/lowvia/issues/new)!

## Setting up your environment

1. Clone the repository: `git clone https://github.com/emanuelVINI01/lowvia.git`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

Make sure you have Ollama installed and running locally with at least one model pulled (e.g., `ollama run llama3`), as the assistant connects to it.

## Submitting a Pull Request

1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure your code lints (`npm run lint`).
5. Issue that pull request!

## Code Style

Please run the linter before submitting your PR to ensure your changes adhere to the project's coding standards.
